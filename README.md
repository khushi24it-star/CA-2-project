Best Selling Books Data Analysis (2023–2025)

Introduction
This project focuses on analyzing the Best-Selling Books dataset (2023–2025) using Python.
The aim is to apply various data preprocessing, transformation, exploratory data analysis,
feature engineering, and text processing techniques.
Dataset Description
The dataset contains information about best-selling books such as title, author, genre,
publication date, print length, and reviews count.
The dataset was sourced from Kaggle.

Techniques Used
1. Handling missing values
2. Removing duplicate records
3. Handling inconsistent data
4. Renaming columns
5. Data type conversion
6. Feature scaling
7. Data normalization
8. Mean, Median, Mode
9. Aggregation
10. Histogram
11. Box plot
12. Bar chart
13. Scatter plot
14. One-hot encoding
15. Label encoding
16. Feature extraction
17. Feature scaling
18. Dimensionality reduction (PCA)
19. Text lowercasing
20. Tokenization

Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
 How to Run
1. Download the repository
2. Open the `.ipynb` file in Jupyter Notebook or Google Colab
3. Run all cells sequentially
